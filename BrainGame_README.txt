BRAIN GAMES pour Polygames, Socièté technique de Polytechnique Montréal.

Le concept est de trouver "Pinky" dans un cerveau. Malheureusement, il y a des méchants villains
monstre rouge (bactéries ? Anticorps ?) qui viennent vous déranger dans votre quête.

Le jeu possède 1 map principale.  Elle est pas très grande mais ca peut vite devenir difficile voir
infaisable. >:D

Le mode multi est boguer quand on fini la partie (les personnages se jette dans un mur). 
Espace permet de traverser les murs, mais ne vous sauve pas des ennemis. 

Hack : F1 en partie jeu pour avoir accès au menu. F2 pour retirer le menu et continuer de jouer
Certains bug avec les murs/ennemis

Controle : 
          Fleches pour joueur 1
	  WASD pour joueur 2

CONFIGURATION (IMPORTANT !!!!) : 1024*768 windowed SVP, sinon les GUI ne scale pas avec la fenetre

************SPOILERS***********************

-Pinky se déplace dans les maps !!! 
-Il y a une BONUS map. Pour l'avoir, il faut finir le mode LEGENDARY !*
-Le mode multi possède deux types de bonus, vitesse et éclairage. Pour gagner
 il faut trouver Pinky avant l'adversaire.

Attention, vous pouvez ne pas bouger a coté d'un ennemi qui passe, il ne vous remarquera pas...
sauf si vous tentez de bouger alors qu'il est proche. Il peut même vous passer dessus, pas grave
il ne fera rien !
